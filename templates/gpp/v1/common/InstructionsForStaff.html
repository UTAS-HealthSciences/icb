Instructions for staff

Who this document is intended for

This document is designed to assist unit content developers in using web pages to deliver their content. No knowledge of web technologies is required; the instructions are provided below and, once mastered, creating and maintaining unit content should be as simple as updating Word documents.

 

How to use this document

This unit has been set up with some resources to help you start creating unit content.

  • There is a content template that provides the framework for creating a new page, meaning that you don’t need to worry about how the page is presented. Using this template allows you to present content that is engaging, useable, and consistent across the unit.
  • There is a components page where you can simply copy and paste content components into your page. Components include headings, images, tables, and activity boxes, for example. You can access the components page (opens in a new tab or window) at any time you require it.

Once you have read and understood the instructions on this page, you may begin converting existing material to present in the content template, or you can create new material afresh.

Who to contact if you need help

The Digital Innovation Team designed and developed the content template and help resources, and are willing to assist you with any problems you encounter while developing and maintaining your online unit after you’ve read these instructions. Contact Digital.Team@utas.edu.au in the first instance. 

To create a new page based on the template

When you are ready to insert a page based on the template into the Content:

  1. In the toolbox, go to Content
  2. Click the New button and select Create a File.  This will open the content editor page.
  3. Click the Browse for a Template button near the top of the editor page.
  4. Browse to the file template > template.html, and select it by clicking the button next to the file. Click the Add button. The template should appear in the content editor.
  5. Click the Change Path button at the bottom of the editor page, and select a sensible place to store your new file, preferably within the Modules folder or a subfolder within. Click Select Path.
  6. In the text box with the words ‘Enter a Title’, give the page a sensible name as you would for a file on your computer, i.e. a short name with no spaces, such as "Module1".
  7. Click the Save as Draft button to complete the process.


Enter your content into the new page

You now have the bare bones of a new page, and you are ready to populate it with new or existing content.

For convenience, the content template does provide common components that you can modify or delete without having to refer to the components page. If you want to add new components, copy then from the components page.

  1. Delete the components that you wish to avoid using for this page. Ensure that you have selected the whole component (including the blank line above and below the component).
  2. Change the names of headings to something appropriate.
  3. For the remaining components, enter new text over the text that begins with ‘lorem ipsum’. If you are pasting text from Microsoft Word, you will need to use the Paste from Word option:
    1. Make sure that your cursor is in the place that you wish to insert the text.
    2. Click the Paste menu by clicking on the down arrow beside the  icon.
    3. Choose the Paste from Word option.
    4. Paste your content into the pane that appears.
    5. Click Update to insert your content.
  1. When you have finished editing the page, choose the Save and/or Publish button to save your work.
  2. Give this page a friendly title by changing the text that you entered as a file name (it can have spaces now).

To insert new components from the components page:

  1. Copy and paste elements from the components page into your new page. Make sure that you copy the blank line above and below the element you are copying.
  2. Once you are certain that you have completely finished editing the page, you may remove any additional blank lines between boxes, paragraphs headings, etc. if you wish. Once those blank lines are removed it can be tricky to insert new content between components.
  3. You can enter text directly in these additional components if you wish (you don't have to paste it from 'Word').

Here are some tips to ensure that you create attractive and usable pages:

  • The MyLO editor includes buttons and menus to easily apply headings, bullet points, italics and boldface etc. into your document.
  • Please use headings levels to create headings and sub-headings etc., rather than modifying font-sizes and styles.
  • Use heading 4 & Heading 5 inside boxes (such as the "Reading" box).
  • You can create links to (existing) Assignment Dropboxes, Discussion boards, Quizzes, Surveys and other content using the Quicklink button: , and browsing to the activity that you wish to link.

Inserting links

Unfortunately you can't just paste a link directly into these pages - it will look like it's worked, and may even colour the link as blue, but it won't actually work. Instead you need to manually insert each link as a Quicklink.  

  • Click the Quicklink icon , and select the Url option.  
  • You can insert the web address and the displayed name of the link in the dialogue box that opens.
  • You should also select the New Window option.

Inserting pictures

Firstly, copy one of the three picture layouts (image centred; left with text to right; or right with text to left) from the components page, complete with the sample image and the surrounding sample text, into the place you want it in your page.

To replace the sample image with your image: 

  1. Click on the sample image
  2. Click on the insert image button in the editor menu () 
  3. Browse to or drag and drop your image file into the file uploader section.
  4. Click the Choose Destination button, and select the Images folder (or a module folder if that makes more sense). Click on the Select a Path button.
  5. Click the Add button.
  6. Supply alternative text (for blind people), and click OK.
  7. You can re-size the image by clicking on it and dragging the corners.

You can then replace any sample text (“Lorem ipsum”) with the text that you want.

You can supply an image caption, or just delete it if it is not required.

Inserting Echo and YouTube Videos

The placeholder Echo or YouTube video can be copied from the components page into your page. Instructions on how to edit your copy to point to your video are included in that document, along with links to an FAQ page with step-by-step instructions. You may require assistance from the Digital Innovation Team to achieve this.

Some notes about pasting EndNote references from MS Word into this template

  • In-text citations inserted into MS word documents that are then pasted into this document should work correctly
  • In Word, if one clicks on an in-text citation, the reader is taken to the corresponding citation in the reference/reading list. Currently the corresponding EndNote bookmarks are not copied from Word when references are pasted in to MyLO. We are working on a simple way to solve this problem, but may need to fix these links manually in the short term. Alternatively a student can just scroll to the bottom of the page to find the corresponding reference.

Printable version

You may notice that there is a 'PDF Version' link near the top of this document. This link will allow students (or staff) to retrieve an automatically generated PDF version of this document that is suitable for printing and use offline. This solution is not yet implemented but will be automatically applied to all pages using this template in mid-2014. Do not delete this link.

 

If something goes wrong with the page

Dont' Panic.

Sometimes (especially if you delete something, including an innocent looking blank line or space) the layout of the page goes awry.

Contact the Digital Innovation Team if this happens.

Remember to save your document frequently.

Some guidance for content structure

  • All modules should start with a list of learning objectives.
  • Please use the Reading, Discussion, Question, and Task boxes as appropriate. These can be copied from the components page.
  • Please paste reading and reference lists (bibliographies) into a 'Reading' Box at the bottom of your document.
  • After each reference is pasted into the reference list at the end of the document, please copy the "[Library CAMxxx Reading List]" or "[Library CAMxxx Text List]" link (as appropriate) at the end of the reference. You can copy the links included at the bottom of the template page. Don't forget to delete the links form the bottom of the template when you are ready publish your page.
