    Assessment

    Assessment Task

     

    Assessment Task 1: Quizzes

    Task description

    Three (3) online multiple choice quizzes relating to ‘the brain’, ‘the diseases’ and ‘the person’ (20 questions each)

    Task length

    Quizzes will be multiple-choice and completed online and thus a word limit is not applicable.

    Links to unit’s intended learning outcomes

    1, 3

    Assessment criteria

    Correct responses to multiple-choice questions.

    All quiz questions will be single attempt format and each quiz will have a completion time limit of 20 minutes.

    Submission requirements

    Ensure you formally submit your quizzes through MyLO by the due date.

    Contribution to final assessment

    20%

    Date due

    11.59pm, Friday 14th February 2014

    echo

    Embedding Echoes: Warning

    Strange things happen if you embed more that one video on a page, the videos that actually display end up being randomly drawn from your course, rather than then ones you specify.

    Only embed a single video on page using the embed code below.

    To embed your video you will need to edit the html code for the document:

    • In echo: copy the link after the "Available URL:" label on the echo page for the recording.
    • In your MyLO page, in edit mode, click on the "</>" icon at the bottom of the page.
    • Find the video by looking for the line that starts with:
        <div class="video echo">
    • Wherever you see the link 
      https://mymediaservice.utas.edu.au:8443/ess/echo/presentation/3e0e549f-0a28-49fe-85dd-9923f4d69412
      replace it with your link, with the exception of the download link.
    • For the download link, replace the link but make sure it ends with "/media.avi?downloadOnly=true".
    • Save the changes to the code.
    • Save/publish the page.

    For an annotated set of instructions and screen-shots go to the BDC Wiki - Embedding Echo Videos in MyLO.
