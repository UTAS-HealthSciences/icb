'Copy to Clipboard' stopped working?

Recent updates to both Chrome and Flash may cause the 'copy to clipboard' button to stop working for you.

This can be fixed relatively easily.

For infomation on the bugs and how to fix them see the Copy to Clipboard bug page.

Note that this message shows all the time. If you can see this message it doesn't mean that the button has stopped working for you. Use the menus above to choose a component to check if the button still works for you.
