Creating pages based on the template

  1. Open the content tool and navigate to the folder (module) in which you want the new page to be placed;
  2. Choose New-> Create a File;
  3. In the file creation tool click the ‘browse for template’ button;
  4. Select the dot next to the ‘template.html’ file in the ‘template’ folder and click ‘Add’;
  5. Enter a Filename WITHOUT SPACES in the ‘Enter a Title’ box;
  6. Click the ‘Change path’ button at the bottom of the editing window, and select a sensible place to store your new page – preferably in the ‘modules’ folder or a sub-folder thereof;
  7. Click ‘Publish’ or ‘Save as Draft’ to save the new file;
  8. Change the Page name to something sensible that a human can read (ie. add spaces back in to the title box).

Editing pages and adding content 

Part A Open the 'Components' page

Start by opening the 'template components' link in a new page or window:

  • From the 'instructions for staff' folder right-click (windows) or context-click (mac) on the link and select 'open in a new tab or window'.
  • Alternatively you can use the link in any new template page you create, but note that the link is only clickable when you are not in edit mode.

Warning: Always copy and paste to and from the same type of web-broswer

Make sure that you use the same type of browser to open the components page as you use to edit your pages in MyLO.  Copying and pasting between different types of browser (eg Firefox-> Chrome, Internet Explorer -> Firefox etc) can cause some weird things to happen.


Part B Add content

Edit the page as per normal, by clicking on the 'Edit HTML' button at the bottom of a page you want to edit.

  • You can replace the existing placeholder text (Lorum ipsum dolor...) that is in the page with your own text, images, links or template components.
  • To paste a new component into your page go to the components page (which you opened in another tab or window), select the component you want from the dropdown menus and click the ‘copy to clipboard button’
  • Paste into your page.
  • Don’t forget to save your page regularly.

Each section of the components page includes information and instructions on how to use those components.

Warning: Do NOT edit the your pages via the 'Manage Files' tool

MyLO allows you to edit your files directly from the 'Manage Files' tool. However MyLO does not correctly save hyperlinks and file paths correctly when files are editied from the Manage Files tool.  Consequently files that use the template will not display correctly if edited via the Mangage Files tool.

Always edit files via the 'Content' tool instead.

Inserting links and images

Links

You can't just paste links from documents or web-sites directly into the MyLO editor. Sometimes this will work, sometimes it won't. Sometimes it will look like it's worked, but won't have actually worked, which can lead to lots of confused students (and staff).

Whenever you need to add a link to a document use the 'quicklink' button () from the MyLO editor toolbar and choose the 'URL' option.  It is usually a good idea to choose the 'open link in a new window' option.

Images

Adding images can be a little tricky

Start by copying the left/right/centred or interactive image template component into your document where you want it, complete with the placeholder image, just as you would for any component.

Click on the sample image so that it is highlighted

For example:

putCaptionHere

Step 3:
While the image is still highlighted click on the insert image icon () in the MyLO editor toolbar, and browse to/upload/drag-&-drop your image. BEFORE you 'Add' your image, make sure you save your image file in a sensible place by clicking on the 'Choose Destination' button, and selecting a folder (eg: the 'images' folder), rather than just the main MyLO File Manager directory.

Step 4:
Supply alternative text for the image, or tick the "this image is decorative" box. 

Step 5:
Overwrite or delete the placeholder caption.

You can re-size the image by clicking on it, and dragging the 'image-handles'.

Copying content from Microsoft Word

MyLo Paste Options Menu

Be careful when copying content from Microsoft Word and other applications. Microsoft Word in particular adds much of its own non-standard custom styles to text which can prevent the template styles from appearing and can interfere with page layout.  If you want to paste text from a Word document, you can use the "Paste as PlainText" option from the dropdown paste options menu in MyLO editor Tool bar.

If you have complex formatting in a Microsoft Word document that you want to keep (such as bullet points, or a table), then you can use the "Paste from Word" option from the same menu. In this case you will need to paste your content from Word into a popup window, which will then be converted to html and put into your page.

Hint

The template contains lots of hidden code to ensure that items display correctly. Whenever your edit a page that uses the template, it's a good idea not to completely delete any of the placeholder text ("Lorem ipsum dolor...") or images, until you have entered some of your own text into the same place.

Deleting the entire text or image might result in more than just placeholder being removed from the document, but may also remove the 'place' into which you wish to enter your text or image.

It can be a good idea to insert your new text (or image) into the middle of the existing text, so that the correct font etc is used and the text is entered into the place you intend.  You can then delete the extraneous placeholder text either side of the text you have written:

For example to replace the 'Lorum ipsum ...' placeholder text with 'This is your new text' you could:

  • Type or paste your content into the middle of the placeholder text
    • Lorum This is your new text ipsum ...
  • Delete the extraneous text:
    • This is your new text

Similarly, If you are pasting a component into a page, it's often a good idea to insert an empty paragraph first, and then paste the component into that space.

Be careful when you delete anything (including extraneous placeholder text or images), as you might also delete part of the underlying webpage code following or preceding the placeholder text, resulting in a strange layout. It is often 'safer' to use the 'delete' key to delete characters etc in front of the cursor, than to use the 'backspace' button to delete characters that are behind the cursor.

If something goes wrong with the page

Don't Panic

Sometimes (especially if you delete something, including an innocent looking blank line or space) the layout of the page goes awry.

You can try "undoing" you last change, or if that doesn't work, close the page without saving it and then re-edit it.

If you have saved the changes, you can contact your school educational developer or technologist who may be able to fix the code so that you don't loose your changes.

For staff in the School of Medicine, you can contact the Digital Innovation Team at Digital.Team@utas.edu.au for help.

Remember to save your document frequently.
