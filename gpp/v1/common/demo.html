Instructions for embedding echo videos

To embed your video you will need to edit the html code for the document:

  • In echo: copy the link after the "Available URL:" label on the echo page for the recording.
  • In your MyLO page, in edit mode, click on the "</>" icon at the bottom of the page.
  • Find the video by looking for the line that starts with:
      <div class="text-center echo">
  • Wherever you see the link 
    https://mymediaservice.utas.edu.au:8443/ess/echo/presentation/3e0e549f-0a28-49fe-85dd-9923f4d69412
    replace it with your link, with the exception of the download link.
  • For the download link, replace the link but make sure it ends with "/media.avi?downloadOnly=true".
  • Save the changes to the code.
  • Save/publish the page.

For an annotated set of instructions and screen-shots go to the BDC Wiki - Embedding Echo Videos in MyLO.
