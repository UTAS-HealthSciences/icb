Instructions for installing the template in your unit

Part A: Get the template

  1. Download this file: DIT-Template-V3.zip, and save it to your computer.

Part B: Install the template in your unit

  1. Go to your unit in MyLO
  2. From the MyLO toolbar start the 'Edit Unit > Import / Export / Copy Components' tool
  3. Use the import option to import the Zip file you downloaded.
    This will load the necessary files into the MyLO file Manager for your unit, and create an 'Instructions for staff' folder in the MyLO Content.  That folder is set to draft so that only staff can see it.  The folder has links to the template components, instructions on using the template, and a demonstration page.

Part C: Adjust your unit code/unit name in the template.

  1. From the 'Content' tool, NOT the 'Manage Files' tool, open the 'template' page, and press the 'Edit HTML' button at the bottom of the page.

    Warning: Do NOT edit the your pages via the 'Manage Files' tool

     

     

    MyLO allows you to edit your files directly from the 'Manage Files' tool. However MyLO does not correctly save hyperlinks and file paths correctly when files are editied from the Manage Files tool.  Consequently files that use the template will not display correctly if edited via the Mangage Files tool.

    Always edit files via the 'Content' tool instead.

  2. Edit the unit code (ie replace the 'DIT000' text).
  3. Edit the unit name (ie replace the 'Digital Innovation Team Template' text). 

    Hint

     

     

    The template conatins lots of hidden code to ensure that items display correctly. Whenever your edit a template it's a good idea not to completely delete any of the placeholder text ("Lorem ipsum dolor...") or images, until you have entered some of your own text into the same place.  

    Deleting the entire text or image might result in more than just placeholder being removed from the document, but may also remove the 'place' into which you wish to enter your text or image.

    It can be a good idea to insert your new text (or image) into the middle of the existing text, so that the correct font etc is used and the text is entered into the place you intend.  You can then delete the exrtaneous placeholder text either side of the text you have written:

    For example to replace the 'DIT000' text with 'ABC123' you could:

    • Type your new unit code into the middle of the old code:
      • D   ABC123   IT000
    • Delete the extraneous text:
      • ABC123

    Be careful when you delete anything (including extraneous placeholder text or images), as you might also delete part of the underlying code following or preceeding the placeholder text, resulting in a strange layout.

  4.  Do not edit anything else!
    Any edits you make will apply to all files that you subsequently create based on your template.

  5. Save the template. 
    Use the "Save" or "Publish" button
    Do not change the file name
    Do not change the location.

  6. Close the template page

Remove the template from the Contents.

  1. Navigate back to the "Instructions for staff" folder in the contents, (not the file manager!)
  2. Click on the triangle next to the "template" page in the contents
  3. Choose the "Delete" option
  4. Choose the FIRST option from the popup dialogue:
    "Remove the topic from Content but keep the associated file or activity in the unit."
    Do NOT choose the second "Permanently delete" option

The template is now ready for use in your unit

Refer to the Instructions for staff to see how to create pages based on the template

If you need to adjust or customise other parts of the template, such as the background or logos, or you wish to collaborate on the development of customised tools for your school or faculty, please contact the Digial Innovation Team at Digital.Team@utas.edu.au
